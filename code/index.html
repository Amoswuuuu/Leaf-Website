「发现」视图

「发现」视图由不同的模块和信息流构成，你可以在信息流中查看最近Leaf+社区里小伙伴们更新的的信息。(1)

0 顶部模块

「Leaf+」的顶部模块，由“搜索 icon”、“Leaf+ logo”和“新建 icon”构成，两个icon分别连向两个页面。

1 标签模块

在标签模块中，小伙伴们可以点击任意一个标签直达你想看到的内容。

2 木叶模块

正所谓“一木一叶，一言一情”，每一个卡片都展示了社区内小伙伴们所发送每一条消息，在这个卡片，你还可以点赞和评论。

3 导航模块

「Leaf+」的导航模块，由当前所在的“发现”页、左侧的“热门”页和右侧的“我的”页组成，小伙伴们可以点击到达不同页面。

登　　录

「Leaf+」的登录系统数据全部源自学校的教务系统。

所有在校生可以凭借学号+学分制系统密码登录「Leaf+」 。这样既保证了「Leaf+」社区的封闭性.同时也保护了社区内小伙作们的隐私。

担心输入数据是否安全?

「Leaf+」的所有数据均是基于HTTPS传输协议的SSH通道向后端传输，并且在后端经过MD5、base64等方案进行加密，服务器与数据库均部署在国内顶级云服务供应商——阿里云，同时后端具有数据备份、数据恢复功能。以上这些都只为保护每一位小伙伴们的数据安全！

搜索与新建

「Leaf+」2.0中提供了搜索功能，同时在原有“新建信息”的基础上增加了上传图片、是否匿名等功能。(1)

我可以搜索哪些内容？

目前「Leaf+」支持标签搜索、关键字搜索等文字信息搜索功能。

为什么我发布的信息看不到？

如果您在发布信息是提示“已提交成功”但是您未在信息流中看到您的信息，您的信息可能被系统过滤掉了，原因是：我们后端采用了接入“阿里云反垃圾”接口 + 自建“敏感词词典”的过滤功能。
